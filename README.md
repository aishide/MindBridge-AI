# 🌉 MindBridge AI

MindBridge AI is a mental health analysis platform that predicts user wellness based on behavioral and psychological inputs.

## 🚀 Features
- Mental health risk prediction (Low / Medium / High)
- Wellness score generation
- Dashboard with charts (Bar, Pie, Line)
- MongoDB data storage
- Interactive UI with React

## 🛠 Tech Stack
Frontend:
- React.js
- Recharts

Backend:
- Flask (Python)
- Machine Learning (Scikit-learn)

Database:
- MongoDB
